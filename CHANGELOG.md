# Changelog

All notable changes are documented here. This project follows semantic versioning.

## 0.3.0 — 2026-07-14

- Ported applicable career-ops 1.19.0 capabilities into RFP-native budget reliability, freshness and issuer exclusions, pursuit-funnel metrics, configurable extraction limits, and proposal metric validation.
- Added private local RFP evaluation through Ollama and Qwen with explicit firm-context loading, loopback privacy protection, output validation, and a bounded summary-repair pass.
- Added guarded double-click launchers with tested Codex + local Qwen agent mode, an experimental Claude Code integration, and plain Qwen chat.
- Added Google Antigravity as an independent launcher option for both Consulting Ops and Career Ops; Career Ops also exposes its official updater from the launcher.
- Audited career-ops 1.20.0 and recorded canonical capability matching plus redirect revalidation as requirements for future targeted capability-gap analysis; excluded career-only manifesto/community automation.
- Documented which upstream job-search features were intentionally not carried into consulting-ops.

## 0.2.0 — 2026-07-11

- Added the installable `consulting-ops` CLI and one-command auto-pipeline.
- Added PDF, DOCX, HTML, Markdown, text, and URL capture/extraction.
- Added amendment handling, proposal export, deadlines, follow-up, research, debrief, and outcome analysis.
- Added tracker verification, normalization, deduplication, reconciliation, liveness, inbox, statistics, search, and quality checks.
- Added provider plugins and safe plugin audit/install tooling.
- Added portable AI CLI adapters, parity/release audits, clean-install validation, and cross-platform CI.

## 0.1.0

- Initial RFP discovery, evaluation, proposal workspace, tracker, and dashboard release.
