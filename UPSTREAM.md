# Upstream attribution

`consulting-ops` began as a domain rewrite of architectural ideas and code from
[`santifer/career-ops`](https://github.com/santifer/career-ops), which is licensed
under the MIT License.

The original project's local-first Markdown data, multi-agent-tool adapters,
pipeline utilities, and dashboard structure informed this work. The consulting
procurement data contract, bid/no-bid workflow, RFP discovery, compliance model,
proposal workspace, and consulting-specific user experience are being developed
for `consulting-ops`.

The original copyright and permission notice remain in `LICENSE`. Git history or
release notes should continue to make the upstream relationship clear.
