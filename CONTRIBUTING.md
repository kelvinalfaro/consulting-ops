# Contributing

Contributions should preserve the cross-AI core and the source-grounded, human-submission safeguards described in `AGENTS.md`.

1. Open an issue describing the user need and proposed data-contract impact.
2. Add or update focused tests.
3. Run `npm test` and `npm run doctor` with neutral example data.
4. Do not commit real firm profiles, confidential RFPs, generated proposals, credentials, or unsupported example claims.
5. Keep vendor-specific adapters thin; portable Node/YAML/Markdown behavior belongs in the core.

By contributing, you agree that your contribution is licensed under the repository's MIT License.
